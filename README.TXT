Este es un programa en consola para idnicar si una palabra es un palindromo

Para ejecutar el ejercicio puede:

Abrir una consola ubicada en la raiz del proyecto y ejecutar el archivo ejercicio2.jar
    java -jar ejercicio2.jar

o

Puede agregar el proyecto al IDE y ejecutarlo desde ahi como Java Aplication

Al ejecutarse le solicitara ingresar una palabra, ingresela y presione enter
el programa le mostrara un mensaje indicando si es palindromo o no y dara la opcion
para que ingrese una nueva palabra
